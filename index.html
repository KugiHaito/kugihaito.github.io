Sobre Min

Desde pequeno, eu sempre fui criativo e cheio de ideias, não como hoje, pois tinha pouco comnhecimeto. Na escola aprendia varias coisas, com tempo fui percebendo o mundo a minha volta, diria que minha infância não foi normal, e isso é bom, nada melhor do que ser diferente.

Ao começar o Ensino Medio, estava focado em ser um Desing Gráfico, mas tudou mudou quando vi um computador, daqueles antigos, pra min era ultra-geração, até hoje me lembro desses dias como se fossem decádas e decádas atrás.

Como toda criança eu queria saber mexer naquela maquina, mas eu sabia que ficar parado de frente a ele, não saberia o que fazer, daí comecei a observar, todos que a utilizavam, não importa para o que fosse. Apartir daí tracei uma meta aprender cada vez mais sobre computação, apesar de gostar mais da parte lógica, me matenho de cabeça aberta.

Agora estou aqui! Programando e estudando cada vez mais, tendo ideias ,e as observando-as, tentando entender o que pode dar de errado, e tudo isto com uma xicará de café na mão.

AppLogin

Aplicação de login e cadastro de usuários em python(2.7 ou 3.4+), com banco de dados pyMySQL.
Este projeto está disponivél na versão beta

Ver em Github Baixar ZIP

Usar

[test mod]
baixe o ZIP ou clone o nosso repositório
git clone https://github.com/KugiHaito/AppLogin.git
Se baixou o ZIP extraia os arquivos gunzip AppLogin-master.zip
Agora instale algumas depêndencias
pip install pymysql
Imorte o Banco de dados
mysql -u root AppLogin < config.sql
Pronto! Agora é só rodar
python -u main.py
